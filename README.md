# front-vue

初学Vue时为了练习Vue全家桶并加深对Vue的认识而做的小项目

# 功能
- [x] 展示商家列表
- [x] 餐馆食品列表页
- [x] 购物车功能
- [x] 店铺评价页面
- [x] 单个食品详情页面
- [x] 商家详情页
- [x] 登录、注册
- [x] 修改密码
- [x] 个人中心页面
- [x] 下单功能
- [x] 订单列表
- [x] 帐户信息
